# **App Name**: Verdant Habits

## Core Features:

- Habit Grid: Display a scrollable horizontal grid organized by weeks, with daily tracking via green checkboxes.
- Progress Calculation: Automatically calculate the 'Monthly Progress' by summing all checked boxes across all habits.
- Dynamic Summary Columns: Display tasks completed (fraction and percentage) and a visual progress bar for each habit.
- Trend Analysis: Track daily completion rates across the month with an area chart.
- Date Display: Display 'Today's Date' in a prominent box.
- Gamification - Reward Section: Display plant/cactus icons in the reward section based on daily streak milestones.
- Local Storage Persistence: Ensure all habit tracking data is saved to the browser's local storage.

## Style Guidelines:

- Primary color: Forest green (#386641) to evoke a sense of growth and nature.
- Background color: Crisp white (#F5F5F5) for a clean, professional aesthetic.
- Accent color: Deep purple (#702963) for the progress bars, providing a strong visual contrast and highlighting progress.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, neutral, and objective look.
- Use Lucide-React icons for checkboxes and UI elements, maintaining a sharp and modern look.
- Mimic a high-end spreadsheet dashboard with a clean and professional layout.
- Use a CSS Grid or HTML Table with sticky positioning for the left-hand 'Habit Name' column and the right-hand 'Total' columns to ensure they remain visible while the central date grid scrolls horizontally.
- Subtle transitions and animations for checkbox interactions and data updates to provide a smooth user experience.
